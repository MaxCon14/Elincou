Foundation

Brand

Elincou connects laboratories, healthcare professionals, researchers and institutions with the technologies they need to operate with confidence. Every design decision should express expertise, reliability and forward-thinking progress.

Vision

To become the trusted scientific partner connecting healthcare, research, and laboratory communities with the technologies that advance progress.

Mission

To support scientific and healthcare advancement by providing access to innovative technologies, expert guidance, and reliable long-term partnerships.

03

04

Logo

Brand

The identity combines a contemporary wordmark with a spherical symbol — a connector of knowledge, technology and scientific advancement. Flowing horizontal lines create movement and information exchange; the spherical form communicates global perspective and long-term vision.

Usage rules 02.5

  • DO Minimum size: 20px height for every identity element
  • DO Clear space: keep the height/width of the symbol free around the logo
  • DO On dark colours use the light identity; on light colours use dark
  • DO Outside the brand palette, use only the black or white logo
  • DON'T Stretch, distort, rotate or outline
  • DON'T Apply effects, gradients or unapproved colours
  • DON'T Use low-contrast combinations
  • DON'T Change logo placement or recolour individual elements

Design Tokens

Build

Every value in this system ships as a CSS custom property in css/tokens.css. Import the file and build with variables — never hard-code brand values.

/* Elincou core tokens — see css/tokens.css for the full set */
:root {
  /* Colour · primary */
  --el-white:       #FFFFFF;
  --el-ghost-white: #F4F4F9;
  --el-alabaster:   #E8E7E4;
  --el-grey-olive:  #999999;
  --el-onyx:        #0E0E0E;

  /* Colour · extended */
  --el-blue:        #3777FF;
  --el-emerald:     #57BB78;
  --el-mustard:     #FFE156;

  /* Typography */
  --el-font-family:    'Gothic A1', sans-serif;
  --el-weight-light:    300;
  --el-weight-regular:  400;
  --el-weight-semibold: 600;

  /* Spacing — 4px base scale */
  --el-space-1: 0.25rem;  --el-space-4: 1rem;
  --el-space-6: 2rem;     --el-space-8: 4rem;

  /* Geometry — sharp system, no rounded corners */
  --el-radius-sm: 0;
  --el-radius-md: 0;
  --el-radius-lg: 0;
  --el-line: rgba(14,14,14,0.16);
}
